title: Margin collapse
---

Remove one or more margins of an element.

## Remove all margins

<div class="theme__outline">
    <div class="theme__outline--inner u-no-margin">
        This div has no margin
    </div>
</div>

```html
<div class="u-no-margin">
    This div has no margin
</div>
```
